Load Distribution Simulation

Deskripsi
Project ini mensimulasikan dua pendekatan load balancing:

1. Static uneven distribution
2. Dynamic distribution

Cara Menjalankan

* Install Python
* Jalankan file:
  python static_dynamic.py

Hasil

* Static menunjukkan proses menuju distribusi ideal
* Dynamic menunjukkan estimasi waktu optimal

Insight

* Static butuh iterasi
* Dynamic lebih efisien untuk sistem real-time
